Basic Image Editor

A simple command-line tool for editing images using Python and the Pillow library. It allows users to perform basic operations such as resizing, cropping, rotating, flipping, adding text, and applying grayscale filters.

Features -
  • Load Images: Open image files for editing.
  • Resize: Adjust the dimensions of the image.
  • Crop: Cut specific portions of the image.
  • Rotate: Rotate the image by a specified angle.
  • Flip: Flip the image horizontally or vertically.
  • Grayscale: Convert the image to grayscale.
  • Add Text: Overlay text on the image with customizable options.
  • Save Edited Images: Save the modified image to a desired location.

Prerequisites -
  • Python 3.x
  • Pillow library

Installation -
  Install Pillow using the command: pip install pillow

Usage -
  1. Launch the program and follow the interactive prompts.
  2. Load an image by specifying its file path.
  3. Choose editing options such as resizing, cropping, rotating, flipping, applying grayscale, or adding text.
  4. Save the final result to a chosen file path.

Notes -
  • The script defaults to a basic font if arial.ttf is unavailable.
  • Ensure input and output file paths are valid.
  • Multiple edits can be applied before saving the image.
